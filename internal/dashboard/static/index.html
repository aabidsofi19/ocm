Metrics Explained
CSA — Configuration Surface Area

Spec definition: count of configurable parameters + environment variables + resource specifications + replica configurations.

This POC extracts (from Kubernetes YAML): container env entries, container ports, leaf keys under resources, presence of spec.replicas, plus a heuristic count of top-level keys under spec.

Evidence: click the CSA tile to see the individual items that contributed to the latest CSA.

OCM — Composite Score

Definition: a weighted aggregate of normalized metrics. This POC stores the composite OCM score per service per run in SQLite.

Note: the current UI does not show a trendline, only the latest OCM value (and a repo-level average).
